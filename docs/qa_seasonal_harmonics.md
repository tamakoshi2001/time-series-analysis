# 試問メモ：周期成分と調波に関するQ&A

平均パターン（曜日×時刻）と状態空間モデルの調波数（日8・週6）に関する想定問答のまとめ。

## Q1. 週周期はなぜ「曜日×時刻」の積で表すのか？

**A. 週の効果が時間帯に依存する（交互作用がある）から。**

- 曜日だけの $w(\text{曜日})$（7値）だと「土曜は24時間一律に−X万kW」という
  モデルになる。実際の週末効果は、昼間（オフィス・工場停止）は大きく、
  深夜（家庭需要中心）はほぼゼロで、時刻に依存する。
- 形式的には、週周期＝168時間周期の任意のパターンは自由度168を持ち、
  （曜日, 時刻）の関数と1対1対応する。曜日のみ（自由度7）はその特殊ケース。
- 実装上は、先に日周期 $d(\text{時刻})$（全曜日平均の1日の形）を引いた残りに
  対して（曜日,時刻）別平均を取るので二重計上はない。
  - $d$ = 平均的な1日の形
  - $w$ = 各曜日の1日の形が平均からどうずれるか（交互作用部分）
- 図 `report_assets/fig_2day_profile.png` で土日の谷が昼間に集中していることが
  交互作用の実データ上の証拠。

## Q2. 状態空間モデルの調波数はなぜ日周期 k=8、週周期 k=6 なのか？

**A. 近似精度と計算量のバランス＋日周期成分との重複回避。**

- 周期 $s$ の季節成分を完全に表すには $s/2$ 調波が必要
  （日: 12、週: 84）。調波1個につき状態変数が2個増える。
- フルに入れると状態次元 ≈ 190。カルマンフィルタの計算量は状態次元の
  2〜3乗に比例するため、17,520時点×最尤推定では非現実的。
- **日周期 k=8**: 朝の立ち上がり・夕方のピークなど尖った形を捉えるには
  多めが必要。k=8 で波長 24/8=3時間までの構造を表現できる。
- **週周期 k=6**: 波長 168/j は j=1..6 で 168h〜28h、すなわち
  「日をまたぐゆっくりした曜日効果」だけを担当。j=7 で日周期と重複する
  （Q3参照）ため、その手前で切る。
- 結果の状態次元は 1 + 2×8 + 2×6 = 29 で計算可能な規模。

## Q3. 「週の第7調波は日周期と被る」の詳細

週周期成分は sin/cos の和：

$$\gamma^{(168)}_t = \sum_{j=1}^{k} (a_j \cos\lambda_j t + b_j \sin\lambda_j t),
\qquad \lambda_j = \tfrac{2\pi j}{168}$$

調波 $j$ は「1週間にちょうど $j$ 回振動する波」で周期は $168/j$ 時間。

| 週の調波 j | 周期 168/j | 意味 |
|---|---|---|
| 1 | 168h | 週に1山 |
| 6 | 28h | 日をまたぐ変動の最細 |
| **7** | **24h** | **日周期の第1調波と同一** |
| 14 | 12h | 日周期の第2調波と同一 |
| 21 | 8h | 日周期の第3調波と同一 |

- $\cos(2\pi \cdot 7t/168) = \cos(2\pi t/24)$ なので、週の調波 $j=7m$ は
  日の調波 $m$ と数式として完全に同じ波。
- 同じ波を表す状態変数が2つあると識別不能（尤度が係数の和にしか依存しない）
  → 最尤推定が不安定になり、表現力は増えずに計算量だけ増える。
- 厳密には重複するのは $j=7,14,21,\dots$ のみ。間の $j=8..13$ などは
  重複しないが「曜日によって日内の形が変わる」交互作用の細部なので、
  計算量とのトレードオフで省略している。
- 要約：「最初の重複が起きる j=7 の手前で切った」。

## Q4. では日周期を無くして週周期フル（84調波=168状態）にすれば良いのでは？

**A. 理論的には可能（張る空間として包含）だが、実用上の代償が大きい。**

1. **計算量**: フル週周期の状態次元は 1+2×84=169（現行29）。
   カルマンフィルタのコストは次元の2〜3乗なので約 (169/29)³ ≈ 200倍。
   数分の推定が日単位になる。
2. **倹約性の喪失**: 日周期成分は「日内の形は毎日共通」という構造で
   波長3時間の解像度をわずか16状態で実現。週周期だけで同じ解像度を
   出すには j=56 調波=112状態が必要。データの支配的構造（毎日繰り返す
   日内カーブ）を使い回せる分解の方が倹約的。
3. **分散パラメータの縮退**: statsmodelsのfreq_seasonalは1成分につき
   撹乱分散が1つ。実際の推定値は日周期 σ²=0.69（ほぼ決定的）に対し
   週周期 σ²=323（大きくドリフト）と約500倍違う。1本にまとめると
   この異質な滑らかさを共通の分散で表すことを強制される。

要約：「数学的には包含関係にあるが、状態次元169 vs 29の計算量と、
成分ごとに異なる分散（0.69 vs 323）を許せる倹約性から日＋週の分解を採る。
これは構造時系列モデルの定石」。
